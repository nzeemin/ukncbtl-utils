## Sav2BkBin

Command line utility used to convert executable RT-11 SAV file into BK-0010 .bin files.

The utility was created to keep my usual RT-11 based toolchain [porting Stop the Express game to BK-0010](https://github.com/nzeemin/bk0010-stopexpress).

### Usage
```
sav2bkbin <inputfile.SAV> <outputfile.BIN>
